3.What is event bubble?

Event bubbling is the propagation of an event that stops from its origin up to the root element. That is, when a user clicks on a button on the page, the event handler under that button propagates the event to its parent, its parent, its parent, and so on to the root element Html. The event notifies all elements above it that someone has clicked on it. Use the stopPropagation() method to prevent bubbling.

4.What is a callback function and why will you use it?

In JavaScript, callback function is a function that is passed as an argument to second function. The function that receives the callback decides when to execute the callback

Callback function can be written in two ways
1. As Anonymous Function.
2. As a regular function.
It optimizes the code, no need to write the same code multiple times
